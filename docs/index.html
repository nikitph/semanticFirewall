The extractor is not the architecture.

Existing systems can keep doing extraction. Grounding Firewall defines what happens next: typed proposal, deterministic gates, idempotent commit, append-only audit.

Proposal sources vary

Rules, LLM structured output, OpenIE, and humans can all emit drafts into the same runtime.

Commit authority is fixed

The runtime, not the generator, decides whether evidence becomes blackboard state.

Memory stays auditable

Accepted supports and rejected proposals both leave structured traces.

Grounding invariant

No quote, no commit.

Every committed support carries raw character offsets into an immutable source chunk. The quoted text must reconstruct exactly from those offsets.
